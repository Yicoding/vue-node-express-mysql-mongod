# mongodb mongoose

> A simple project use of the vue, node, express,mongodb

## Build Setup

``` bash
# install dependencies
npm install

# 安装完数据库后，启动mongodb(windows下)
mongod.exe

# 新开一个窗口，连接mongodb
mongo.exe

# serve with hot reload at localhost:3000
npm run dev
```
